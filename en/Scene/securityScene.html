1.2 Pull the scene list

Interface Description:

Pull the scene list

- (void)createSceneWithHomeId:(long long)homeId
                   sceneModel:(TuyaSmartSceneModel *)sceneModel
                      success:(nullable AddNewSceneSuccessBlock)success
                      failure:(nullable void(^)(NSError *error))failure;

Parameter Description

parameter Description
homeId Family id
sceneModel Scene entity class
success Success callback
failure Failure callback
sceneModel field Description
sceneId Scene id
code Scene code (only the default scene has this field)
name Scene name
enabl Whether the automation scene is turned on
stickyOnTop Whether to display on the room page
NSArray *preConditions; Scene preconditions (not used by security protection)
NSArray *conditions Scene condition
NSArray *actions; Scene action
devlist Device List
coverIcon The url of the scene icon
displayColor The background color of the scene
subTitle Scene subtitle
editIcon Edit icon
displayIcon Show icon
background Scene background image url
matchType Execution conditions (enumeration: satisfy any condition, satisfy all conditions)
boundForPanel Is there a scene panel binding (not used by security yet)
boundForWiFiPanel Is there a new firmware scene panel binding, this type supports zigbee devices and wifi devices (not used by security temporarily)
localLinkage Whether to support local linkage (not used by security temporarily)
recommendType Recommended scene type (not used by security yet)
collectionType Collection scene type (not used by security yet)
recomDescription Recommended scene description (not used by security yet)
recomCoefficient Recommended scene recommendation coefficient (not used by security yet)
disableTime The timestamp of the time when the automation scene will be automatically closed. 0 means it has not been set.
TuyaSmartSceneConditionModel field description
conditionId Condition id
iconUrl Condition icon
entityId City ID or device ID, the ID of the entity corresponding to the condition
entityName Corresponding entity class name
entityType Corresponding entity class type (enumeration value)
exprDisplay Condition display content
scenarioId Scene id
entitySubIds Condition or dp point id
expr Conditional expression
status Condition status
extraInfo Condition entity class specific information
cityName city name
cityLatitude City longitude (not used by security temporarily)
cityLongitude City dimension (not yet used by security)
productId Recommended product id (not used by security yet)
productPic Recommended product icon (not used by security yet)
TuyaSmartSceneActionModel field description
actionId Action id
actionDisplay The description of the action to be done is such as: "Switch: Off".
entityId City ID or device ID, the ID of the entity corresponding to the condition
entityName Corresponding entity class name
actionExecutor Task execution attributes
executorProperty Execute attribute value
status Execution status
uiid Panel id (not used by security yet)
productId Recommended product id (not used by security yet)
productPic Recommended product icon (not used by security yet)

Code example

[[TuyaSecuritySceneManager sharedInstance] getSceneListWithHomeId:123 success:^(NSArray<TuyaSmartSceneModel *> * _Nonnull
list) {

} failure:^(NSError * _Nonnull error) {

}];
