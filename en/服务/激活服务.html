激活服务

TuyaSecurityActivationCode 激活码服务

获取用户的服务信息

获取用户绑定的服务商Id，渠道ID，绑定的激活码，并且可以判断出客户是否绑定激活码

如果不需要激活码的操作，直接联系 Tuya 开发开启默认激活码，这样保证每次获取到的值都是绑定过的

参数 说明
success 成功回调
failure 失败回调 

/// get user service information.
/// 获取用户的服务信息
///
/// If the user has already bound the activation code, the current service information is recorded locally
/// 如果用户已经绑定激活码，当前服务信息会记录在本地
/// @param success Success Block
/// @param failure Failure Block

- (void)getServiceInfoWithHandler:(GetUserServiceHandler)success
                          failure:(TYFailureError)failure;
返回值 类型 说明
isBind NSInteger 是否绑定激活码
code NSString * 激活码
channelId NSString * 绑定的激活码对应的渠道ID
dealerId NSString * 绑定的激活码对应的服务商ID

代码示例：

- (void)loadDealerInformation {

    [[TuyaSecurityActivationCode sharedInstance] getServiceInfoWithHandler:^(TYUserServiceModel * _Nullable userServiceModel) {

    } failure:^(NSError *error) {

    }];

}

绑定服务激活码

绑定激活码

参数 说明
aCode 激活码
success 成功回调
failure 失败回调 
/// Binding service activation code
/// 绑定服务激活码
/// @param aCode Activation code
/// @param success Success Block
/// @param failure Failure Block
- (void)bindServiceWithCode:(NSString *)aCode
                    success:(TYSuccessID)success
                    failure:(TYFailureError)failure;

示例：

- (void)bindDealerWithCode:(NSString *)aCode {

    [[TuyaSecurityActivationCode sharedInstance] bindServiceWithCode:aCode success:^(id result) {
        //绑定成功

    } failure:^(NSError *error) {
        //绑定失败
    }];
}

TuyaSecurityServiceConfig 增值服务配置信息

获取服务配置

/// Get service configuration
/// 获取服务配置
/// @param success success Block，目前有service_url（服务套餐地址） help_url（帮助页面连接）
/// @param failure failure Block
- (void)getServiceConfigWithHandler:(TYSuccessDict)success
                            failure:(TYFailureError)failure;

示例:

[[TuyaSecurityServiceConfig sharedInstance] getServiceConfigWithHandler:^(NSDictionary *dict) {

            } failure:^(NSError *error) {

            }];

获取本地服务配置

/// Configuration based on the get URL of the userId
/// 根据userId的获取URL配置
/// @param userId userId
- (NSDictionary *)getServiceConfigURLMapFromCacheWithUid:(NSString *)userId;

示例：

NSDictionary *configDict  =  
[[TuyaSecurityServiceConfig sharedInstance] getServiceConfigURLMapFromCacheWithUid:[TuyaSmartUser sharedInstance].uid];
