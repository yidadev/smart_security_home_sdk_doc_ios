Preparation for Integration

1.1. Preparation

1.1.1 Registering a developer account

Go Tuya Development Platform registered developer account, create products, create points function, etc. please refer to the specific process Access Procedure

1.1.2 Create SDK Application

Click "App SDK" in "App Workbench" in Tuya IoT platform, and click to get SDK.

Choose a suitable development plan according to your needs and click "Next".

Note: For the security SDK, the IPC scheme needs to be checked here so that the SDK can provide Camera-related capabilities.

Can be obtained in the iOS section AppKey、AppSecret，click on the "Download security image", the security picture Rename t_s.bmp.

Note: For security, do not download the iOS (Podfile), use the version we provide. The specific focus is on the integration of Security SDK.
