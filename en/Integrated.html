Integrate SDK

1.1. Integrate Security SDK

1.1.1 Use CocoaPods to quickly integrate (the minimum SDK supports system version 9.0)

In the Podfile Add the following files:

platform :ios, '9.0'

target 'Your_Project_Name' do
  // Tuya Basical SDK
    pod 'TuyaSmartHomeKit','~>3.17.0'
  // Tuya Camera SDK
  pod 'TuyaSmartCameraKit', '~>4.17.1'
  pod 'TuyaSmartCameraT', '~>4.17.6'
  // Tuua Security SDK
  pod 'TuyaSmartSecurityBaseKit', '~>2.1.0'
  pod 'TuyaSmartSecurityBizAopKit', '~>2.1.0'
  pod 'TuyaSmartSecurityServiceKit', '~>2.1.0'

end

And then execute the project root directory pod update command integrate.

For the use of CocoaPods, please refer to:CocoaPods GuidesCocoaPods is recommended to update to the latest version

1.1.2 Initialize SDK

  1. Open the project settings, Target => General, modifyBundle Identifierit to the iOS package name corresponding to the App registered on the Tuya developer platform.

  2. Import the security image downloaded in the abovePreparations into the project root directory, rename it tot_s.bmp and add it to "Project Settings => Target => Build Phases => Copy Bundle Resources".

  3. PrefixHeader.pchAdd the following content to the project file (Swift project can be added in thexxx_Bridging-Header.hbridge file):

    #import <TuyaSmartBaseKit/TuyaSmartBaseKit.h>
#import <TuyaSmartActivatorKit/TuyaSmartActivatorKit.h>
#import <TuyaSmartCameraKit/TuyaSmartCameraKit.h>
#import <TuyaSmartSecurityBaseKit/TuyaSmartSecurityBaseKit.h>
#import <TuyaSmartSecurityServiceKit/TuyaSmartSecurityServiceKit.h>

  4. Open the AppDelegate.mfile, in the[AppDelegate application:didFinishLaunchingWithOptions:]method, used on graffiti developer platform, App corresponding App Key，App Secret initialize the SDK:

    Objc

    [[TuyaSmartSDK sharedInstance] startWithAppKey:<#your_app_key#> secretKey:<#your_secret_key#>];

​ Swift:

TuyaSmartSDK.sharedInstance()?.start(withAppKey: <#your_app_key#>, secretKey: <#your_secret_key#>)

At this point, Tuya Security App SDK has been successfully activated, and App development can begin.

1.1.3. Debug mode

During the development process, you can turn on the Debug mode and print some logs for problem analysis.

ObjC

#ifdef DEBUG
    [[TuyaSmartSDK sharedInstance] setDebugMode:YES];
#else
#endif

Swift

#if DEBUG
   TuyaSmartSDK.sharedInstance()?.debugMode = true
#else
#endif
