安防场景

场景SDK--iOS

接口说明
初始化
/// 单例初始化
+ (instancetype)sharedInstance;
拉取场景列表
- (void)createSceneWithHomeId:(long long)homeId
                   sceneModel:(TuyaSmartSceneModel *)sceneModel
                      success:(nullable AddNewSceneSuccessBlock)success
                      failure:(nullable void(^)(NSError *error))failure;
参数说明
参数 说明
homeId 家庭id
sceneModel 场景实体类
success 成功回调
failure 失败回调
sceneModel字段 说明
sceneId 场景id
code 场景code(只有默认场景有该字段)
name 场景名称
enabl 自动化场景是否开启
stickyOnTop 是否显示在房间页面
NSArray *preConditions; 场景前置条件(安防暂未用到)
NSArray *conditions 场景条件
NSArray *actions; 场景动作
devList 设备列表
coverIcon 场景icon的url
displayColor 场景的背景色
subTitle 场景子标题
editIcon 编辑icon
displayIcon 显示图标
background 场景背景图片url
matchType 执行条件(枚举：满足任一条件、满足所有条件)
boundForPanel 是否有场景面板绑定(安防暂未用到)
boundForWiFiPanel 是否有新固件场景面板绑定，此类支持zigbee设备和wifi设备(安防暂未用到)
localLinkage 是否支持本地联动(安防暂未用到)
recommendType 推荐场景类型(安防暂未用到)
collectionType 收藏场景类型(安防暂未用到)
recomDescription 推荐场景描述(安防暂未用到)
recomCoefficient 推荐场景推荐系数(安防暂未用到)
disableTime 自动化场景将会被自动关闭的时间点的时间戳。0表示未被设置过。
TuyaSmartSceneConditionModel 字段 描述
conditionId 条件id
iconUrl 条件图标icon
entityId 城市Id或设备id, 条件对应的实体的Id
entityName 对应实体类名称
entityType 对应实体类类型(枚举值)
exprDisplay 条件显示内容
scenarioId 场景id
entitySubIds 条件或者dp点id
expr 条件表达式
status 条件状态
extraInfo 条件实体类具体信息
cityName 城市名称
cityLatitude 城市经度(安防暂未用到)
cityLongitude 城市维度(安防暂未用到)
productId 推荐商品id(安防暂未用到)
productPic 推荐商品图标(安防暂未用到)
TuyaSmartSceneActionModel 字段 描述
actionId 动作id
actionDisplay 动作要做的内容描述 如：“开关 : 关闭”。
entityId 城市Id或设备id, 条件对应的实体的Id
entityName 对应实体类名称
actionExecutor 任务执行属性
executorProperty 执行属性值
status 执行状态
uiid 面板id(安防暂未用到)
productId 推荐商品id(安防暂未用到)
productPic 推荐商品图标(安防暂未用到)

[[TuyaSecuritySceneManager sharedInstance] getSceneListWithHomeId:123 success:^(NSArray<TuyaSmartSceneModel *> * _Nonnull
list) {
    //场景数据

    //场景数据
}];
