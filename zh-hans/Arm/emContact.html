紧急联系人

安防紧急联系人

1.1 初始化
/// 单例初始化
+ (instancetype)sharedInstance;
1.2 拉取紧急联系人
- (void)getLocationEmergencyConnectWithHomeId:(long long)homeId
                                      success:(nullable TYEmergencyConnectListBlock)success
                                      failure:(nullable void(^)(NSError *error))failure;
1.2.1 参数说明
参数 说明
homeId 家庭id
success 成功回调
failure 失败回调
1.2.2 返回数据实体类字段
字段 描述
areaCode 地区编码
areaName 地区名称
phone 联系方式
gmtCreate 创建时间
id 联系人id
sequence 序列
homeId 家庭id
name 姓名
firstName 姓
lastName 名
1.2.3 代码示例
[[TuyaSecurityConnectManager sharedInstance] getLocationEmergencyConnectWithHomeId:1111

success:
(NSArray<TYEmergencyListModel *>
* _Nonnull dataArray) {
    //联系人信息
}
                                                                           failure:^(NSError *
_Nonnull error) {
    //错误信息
}];
1.3 添加紧急联系人
- (void)addNewConnectWithHomeId:(long long)homeId
                   connectModel:(TYEmergencyListModel *)infoModel
                        success:(nullable void(^)(BOOL success))success
                        failure:(nullable void(^)(NSError *error))failure;
1.3.1 参数说明
参数 说明
homeId 家庭id
infoModel 联系人信息(姓名、联系方式、地区编码)
success 成功回调
failure 失败回调
1.3.2 代码示例
[[TuyaSecurityConnectManager sharedInstance] addNewConnectWithHomeId:123
                                                        connectModel:[TYEmergencyListModel new]
                                                             success:^(BOOL success) {
    //成功回调
}
                                                             failure:^(NSError * _Nonnull error) {
    //错误信息
}];

- (void)editConnectWithHomeId:(long long)homeId
                  connectInfo:(NSString *)infoString
                      success:(nullable void(^)(BOOL success))success
                      failure:(nullable void(^)(NSError *error))failure;
1.4.1 参数说明
参数 说明
homeId 家庭id
infoString 联系人信息(json字符串)
success 成功回调
failure 失败回调
1.4.2 代码示例
[[TuyaSecurityConnectManager sharedInstance] editConnectWithHomeId:123
                                                       connectInfo:@""
                                                           success:^(BOOL success) {
    //成功回调
}
                                                           failure:^(NSError * _Nonnull error) {
    //错误信息
}];
1.5 删除紧急联系人
- (void)deleteConnectWithHomeId:(long long)homeId
                 withConnectIds:(NSString *)ids
                        success:(nullable void(^)(BOOL success))success
                        failure:(nullable void(^)(NSError *error))failure;
1.5.1参数说明
参数 说明
homeId 家庭id
ids 联系人信息(格式: "[id]")
success 成功回调
failure 失败回调

                                                      withConnectIds:@""
                                                             success:^(BOOL success) {
    //成功回调
}
                                                             failure:^(NSError * _Nonnull error) {
    //错误信息
}];
