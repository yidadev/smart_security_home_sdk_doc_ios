1 事件绑定

1.1 获取传感设备列表

接口说明

获取传感设备列表

/// 获取传感设备列表
/// Gets a list of sensing devices
/// 
/// @param homeId 家庭Id / Home ID
/// @param success 成功回调 / Success Callback
/// @param failure 失败回调 / Fauilre Callback
- (void)getSensorList:(long long)homeId
              success:(EventRecordingSensorList)success
              failure:(TYFailureError)failure;

参数说明

参数 说明
homeId 家庭Id
success 成功回调
failure 失败回调

事件绑定摄像头的模型 TYEventRecordingBindCameraModel

数据字段 字段类型 说明
entityId NSString * 设备Id
type enum(TYEventRecordingCameraBindingType) 布防类型

事件绑定传感器的模型 TYEventRecordingSensorModel

数据字段 字段类型 说明
deviceId NSString * 设备Id
actionInfos NSArray * 绑定的摄像头列表
linked BOOL 是否绑定过摄像头

代码示例

Objc

[[TuyaSmartSecurityEventRecording sharedInstance] getSensorList:123123 success:^(NSArray<TuyaSmartSecurityEventRecordingSensorModel *> * _Nullable sensorList) {

} failure:^(NSError *error) {

}];

1.2 获取传感器绑定的摄像头列表

接口说明

获取传感器绑定的摄像头列表

/// 获取传感器绑定的摄像头列表
/// Gets a list of cameras for the specified sensor bindings
///
/// @param homeId 家庭Id / Home ID
/// @param deviceId 传感器设备Id
/// @param success 成功回调 / Success Callback
/// @param failure 失败回调 / Fauilre Callback
- (void)getLinkedCameras:(long long)homeId
                deviceId:(NSString *)deviceId
                 success:(EventRecordingCameraLink)success
                 failure:(TYFailureError)failure;

参数说明

参数 说明
homeId 家庭ID
deviceId 传感器的设备ID
success 成功回调
failure 失败回调

事件绑定摄像头的规则模型 TYEventRecordingLinkRuleModel

数据字段 字段类型 说明
ruleId NSString * 规则ID
deviceId NSString * 传感器的设备ID
actionInfos NSArray * 绑定的摄像头列表

代码示例：

Objc

[[TuyaSmartSecurityEventRecording sharedInstance] getLinkedCameras:123123 deviceId:@"sensor Id" success:^(TuyaSmartSecurityEventRecordingLinkRuleModel * _Nullable linkRuleModel) {

} failure:^(NSError *error) {

}];

1.3 绑定摄像头

接口说明

绑定摄像头

/// 绑定摄像头
/// Bind Cameras
///
/// @param homeId 家庭Id / Home ID
/// @param taskModel 绑定摄像头数据的模型
/// @param success 成功回调 / Success Callback
/// @param failure 失败回调 / Fauilre Callback
- (void)linkCamera:(long long)homeId
         taskModel:(TuyaSmartSecurityEventRecordingLinkTaskModel *)taskModel
           success:(TYSuccessBOOL)success
           failure:(TYFailureError)failure;

参数说明

参数 说明
homeId 家庭Id
taskModel 绑定摄像头数据的模型
success 成功回调
failure 失败回调

绑定摄像头的任务模型 TYEventRecordingLinkTaskCameraModel

数据字段 字段类型 说明
cameraId NSString * 设备Id
type NSInteger 绑定类型（1是图片，2是视频）
isSelected BOOL 是否绑定

绑定设备的任务模型 TYEventRecordingLinkTaskModel

数据字段 字段类型 说明
ruleId NSString * 规则ID
sensorId NSArray * 传感器的设备ID
productId BOOL 传感器的产品ID
cameraModelList NSArray * 摄像头列表

代码示例

Objc

TuyaSmartSecurityEventRecordingLinkTaskModel *taskModel = [TuyaSmartSecurityEventRecordingLinkTaskModel new];
    taskModel.ruleId = @"ruleId";
    taskModel.sensorId = @"sensor Id";
    [[TuyaSmartSecurityEventRecording sharedInstance] linkCamera:123123 taskModel:taskModel success:^(BOOL result) {

    } failure:^(NSError *error) {

    }];
