Arm Reminder

1 Arming reminder

1.1 Get the list of arming reminders

Interface Description

Get the list of arming reminders

/// Get ArmReminder list
///
/// @param homeId Home ID
/// @param success Success Callback
/// @param failure Failure Callback
- (void)getArmReminderStatus:(long long)homeId
                     success:(ArmReminderList)success
                     failure:(TYFailureError)failure;

Parameter Description

parameter Description
homeId Family Id
success Success callback
failure Failure callback

Geofencing equipment data model TYArmReminderModel

Data field Field Type Description
homeId NSString * Family ID
type enum(TYArmReminderType) Arming type
enable BOOL Status (open or closed)

Code example

Objc

[[TuyaSmartSecurityArmReminder sharedInstance] getArmReminderStatus:123134 success:^(NSArray<TuyaSmartSecurityArmReminderModel *> * _Nullable armReminderList) {

} failure:^(NSError *error) {

}];

1.2 Modify the arming reminder status

Interface Description

Modify arm reminder status

/// Modify ArmReminder
///
/// @param homeId  Home ID
/// @param type  type
/// @param enable open / close
/// @param success Success Callback
/// @param failure Failure Callback
- (void)updateArmReminderStatus:(long long)homeId
                           type:(TYArmReminderType)type
                         enable:(BOOL)enable
                        success:(TYSuccessBOOL)success
                        failure:(TYFailureError)failure;

Parameter Description

parameter Description
homeId Family Id
type Arming type
enable Status (open or closed)
success Success callback
failure Failure callback

Code example

Objc

[[TuyaSmartSecurityArmReminder sharedInstance] updateArmReminderStatus:124 type:TYArmReminderTypeStay enable:YES success:^(BOOL result) {
} failure:^(NSError *error) {
}];
