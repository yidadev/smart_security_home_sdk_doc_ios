功能概述

Tuya Security SDK 是一套涂鸦安防针对 家庭和商业 安全领域提供的 iOS 9.0 及以上版本的SDK。iOS 开发者可以基于 SDK 快速的实现 APP 功能开发，实现 布防，撤防以及安防场景功能。Tuya Security SDK 以 Tuya Home SDK为基础，做了上层的安防的应用场景，并且集成了Tuya IPC SDK，提供丰富的摄像头能力。

Security SDK主要包括以下内容

  • 安防用户服务（Uid登录，绑定服务码）
  • 布撤防（布防，撤防，报警，取消报警，布防设置）
  • 安防场景（联动场景，地理围栏，布防提醒，事件绑定）
  • 安防消息 （安防布撤防消息，设备消息）
