1.1 上传手机设备信息

接口说明

在登录成功时，收集设备信息给云端


/// 上传手机设备信息
/// Upload mobile device information
///
/// @param homeId 家庭Id / Home ID
/// @param deviceModel 设备信息Model / Mobile phone information
/// @param success 成功回调 / Success Callback
/// @param failure 失败回调 / Failure Callback
- (void)uploadMobileInfo:(long long)homeId
             deviceModel:(TYGeofenceMobileInfoModel *)deviceModel
                 success:(TYSuccessDict)success
                 failure:(TYFailureError)failure;

参数说明：

参数 说明
homeId 家庭Id
deviceModel 设备信息Model
success 成功回调
failure 失败回调

地理围栏设备 数据模型 TYGeofenceMobileInfoModel

数据字段 字段类型 说明
deviceId NSString * 移动设备id
deviceName NSString * 移动设备名称

代码示例：

Objc

TuyaSmartSecurityGeofenceMobileInfoModel *model = [TuyaSmartSecurityGeofenceMobileInfoModel new];
model.deviceId = @"1111111111111111";
model.deviceName = @"testGeofence";

TuyaSmartHome *smartHome = [TYSmartHomeManager sharedInstance].currentHome;
[[TuyaSmartSecurityGeofence sharedInstance] uploadMobileInfo:smartHome.homeModel.homeId deviceModel:model success:^(NSDictionary *dict) {
      NSLog(@"%@", dict);
} failure:^(NSError *error) {
      NSLog(@"%@", error.localizedDescription);
 }];

Swift

let model = TuyaSmartSecurityGeofenceMobileInfoModel()
model.deviceId = "1111111111111111"
model.deviceName = "testGeofence"
TuyaSmartSecurityGeofence.sharedInstance().uploadMobileInfo(231231231, deviceModel: model, success: { (dict) in
     print("dict:  /(dict)");
}) { (error) in
     if let error = error {
           print("show error: /(error))")
      }
}

1.13 手机离开或者进入地理围栏区域

监控设备离开地理围栏的区域，触发进入或者离开的事件

/// 手机离开或者进入地理围栏区域

/// @param homeId 家庭id / Home ID
/// @param geofenceHomeId 触发地理围栏的homeId / The homeId that triggered the geographic fence
/// @param deviceId 手机的唯一标识 / Unique identification of mobile phone
/// @param isLeaving 是否是离家 / Leave home or not
/// @param success 成功回调 / Success Callback
/// @param failure 失败回调 / Failure Callback
- (void)processMobileInfo:(long long)homeId
           geofenceHomeId:(long long)geofenceHomeId
                 deviceId:(NSString *)deviceId
                 isLeving:(BOOL)isLeaving
                  success:(TYSuccessBOOL)success
                  failure:(TYFailureError)failure;

参数说明：

参数 说明
homeId 家庭Id
geofenceHomeId 触发地理围栏的家庭ID
deviceId 手机的唯一标识
isLeaving 是否是离家
success 成功回调
failure 失败回调

代码示例：

Objc

TuyaSmartHome *smartHome = [TYSmartHomeManager sharedInstance].currentHome;
[[TuyaSmartSecurityGeofence sharedInstance]
     processMobileInfo:smartHome.homeModel.homeId
     geofenceHomeId:smartHome.homeModel.homeId
     deviceId:kTestDeviceUUId
     isLeving:NO
     success:^(BOOL result) {
        #ifdef DEBUG
                NSLog(@"reach the home success");
        #endif
    } failure:^(NSError *error) {
        #ifdef DEBUG
                NSLog(@"error information  %@ ", error.localizedDescription);
        #endif
    }];
