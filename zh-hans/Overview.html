功能描述

功能概述

iOS Security SDK 是一套涂鸦安防针对 家庭和商业 安全领域提供的 iOS 9.0 及以上版本的SDK。iOS 开发者可以基于 SDK 快速的实现 APP 功能开发，实现 布防，撤防功能。Security SDK以 Home SDK为基础，做了上层的安全场景。

Home SDK的描述

Security SDK主要包括以下内容

  • 布撤防（布防，撤防，报警，取消报警，布防设置）
  • 安防场景（联动场景，地理围栏，布防提醒，事件绑定）
  • 安防消息
  • 安防服务
