模式设置

1. 模式设置

1.1 获取所有模式的延时规则列表

接口说明：

获取所有模式的延时规则列表

/// 获取指定布防模式下的设备
/// Gets the device in the specified deployment mode
///
/// @param modeStatus 布防模式（在家布防，离家布防）/ arm mode(stay, away)
/// @param homeId 家庭id / Home ID
/// @param success 成功回调 / Success Callback
/// @param failure 失败回调 / Failure Callback
- (void)getDeviceListWithMode:(ArmModeStatus)modeStatus
                       homeId:(long long)homeId
                      success:(GetArmedRuleDeviceList)success
                      failure:(TYFailureError)failure;

参数说明：

参数 说明
homeId 家庭id
success 成功回调
failure 失败回调

数据Model

数据字段 数据类型 说明
mode enum(ArmModeStatus) 布防模式「目前只有在家布防，离家布防」
uid NSInteger 布防延迟时间
ownerId NSInteger 报警延迟时间

代码示例

Objc

[[TuyaSecurityArmedRuleSetting sharedInstance]
     getAllModeDelayRuleWithHomeId:[self getHomeId]
     success:^(NSArray<TYArmedRuleDelayTimeModel *> * _Nullable delayTimeList) {

    } failure:^(NSError *error) {

    }];

获取模式下(在家布防/离家布防)的延迟规则

/// 获取指定模式的延迟规则

///
/// @param modeStatus 布防模式（在家布防，离家布防）/ arm mode(stay, away)

/// @param failure 失败回调 / Failure Callback
- (void)getDelayTimeWithMode:(ArmModeStatus)modeStatus
                      homeId:(long long)homeId
                     success:(GetArmedRuleDelayData)success
                     failure:(TYFailureError)failure;

参数说明：

参数 说明
modeStatus 布防模式（在家布防，离家布防）
homeId 家庭id
success 成功回调
failure 失败回调

数据Model 数据字段 | 数据类型 | 说明 ---|---|--- mode | enum(ArmModeStatus) | 布防模式「目前只有在家布防，离家布防」 uid | NSInteger | 布防延迟时间 ownerId | NSInteger | 报警延迟时间

1.4 获取指定模式下的设备规则列表

接口说明：

获取指定模式下的设备规则列表

/// 获取指定布防模式下的设备
/// Gets the device in the specified deployment mode
///
/// @param modeStatus 布防模式（在家布防，离家布防）/ arm mode(stay, away)
/// @param homeId 家庭id / Home ID
/// @param success 成功回调 / Success Callback
/// @param failure 失败回调 / Failure Callback
- (void)getDeviceListWithMode:(ArmModeStatus)modeStatus
                       homeId:(long long)homeId
                      success:(GetArmedRuleDeviceList)success
                      failure:(TYFailureError)failure;

参数说明：

参数 说明
modeStatus 布防模式（在家布防，离家布防）
homeId 家庭id
success 成功回调
failure 失败回调

数据Model 数据字段 | 数据类型 | 说明 ---|---|--- deviceId | NSString | 设备id uid | NSString | 用户id ownerId | NSString | 家庭拥有者id name | NSString | 设备的名称（配网后设备的名称） customName | NSString | 设备的别称（修改后的设备名称） ability | NSInteger | 设备能力 productId | NSString | 设备产品分类id productCode | NSString | 设备产品代码 categoryName | NSString | 设备类别 sub | BOOL | 是否是子设备 fixed | BOOL | 是否是环境设备（环境设备不参与选择，所有布防类型下都是参与联动） selected | BOOL | 是否选中，安全设备是否参与布防类型的规则 icon | NSString * | 设备的图片 activeTime | NSTimeInterval | 设备的激活时间

代码示例：

Objc


[[TuyaSecurityArmedRuleSetting sharedInstance]
     getDeviceListWithMode:(ArmModeStatus)modeType
     homeId:homeId
     success:^(NSArray<TYArmedRuleDeviceModel *> * _Nullable nDeviceList) {

    }
     failure:^(NSError *error) {

    }];
