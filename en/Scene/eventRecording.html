1.3 Binding camera

Interface Description

Binding camera

/// Bind Cameras
///
/// @param homeId Home ID
/// @param taskModel 
/// @param success Success Callback
/// @param failure  Fauilre Callback
- (void)linkCamera:(long long)homeId
         taskModel:(TuyaSmartSecurityEventRecordingLinkTaskModel *)taskModel
           success:(TYSuccessBOOL)success
           failure:(TYFailureError)failure;

Parameter Description

parameter Description
homeId Family Id
taskModel Model for binding camera data
success Success callback
failure Failure callback

The task model of the bound camera TYEventRecordingLinkTaskCameraModel

Data field Field Type Description
cameraId NSString * Device Id
type NSInteger Binding type (1 is picture, 2 is video)
isSelected BOOL Whether to bind

The task model of the binding device TYEventRecordingLinkTaskModel

Data field Field Type Description
ruleId NSString * Rule ID
sensorId NSArray * The device ID of the sensor
productId BOOL Product ID of the sensor
cameraModelList NSArray * Camera list

Code example

Objc

TuyaSmartSecurityEventRecordingLinkTaskModel *taskModel = [TuyaSmartSecurityEventRecordingLinkTaskModel new];
    taskModel.ruleId = @"ruleId";
    taskModel.sensorId = @"sensor Id";
    [[TuyaSmartSecurityEventRecording sharedInstance] linkCamera:123123 taskModel:taskModel success:^(BOOL result) {

    } failure:^(NSError *error) {

    }];
